## Installation 
### Mise en place
```
    git clone https://github.com/MartinCayuelas/formatech-api
    cd formatech-api
    npm run build
    npm start

```
-------------
    
### Déploiement

  ```
apps:create formatech

proxy:ports-add api-formatech http:80:3000
```
